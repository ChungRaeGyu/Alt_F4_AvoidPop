## 게임 설명
- 하늘에서 떨어지는 똥을 피하며 오래오래 버티는 게임입니다.<br>
- 1인 플레이와 로컬2인플레이를 지원합니다.
- 1인 플레이시 랭킹을 저장하여 최고점수를 나타내 줍니다.
- 2인 플레이시 상대방과 경쟁하여서 누가더 오래 살아남았는지 게임종료시 표시해 줍니다.
